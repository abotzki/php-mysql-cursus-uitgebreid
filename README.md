
# PHP & MySQL Webontwikkeling - uitgebreide cursus

Deze repository bevat een uitgebreid Nederlandstalig cursusboekje, Markdown-hoofdstukken, invulopdrachten en PHP/MySQL-snippets.

## Starten
```bash
cp .env.example .env
docker compose up -d
```
Open daarna `http://localhost:8080`.

## Inhoud
- `boekje/`: Word-versie van het uitgebreide cursusboekje.
- `docs/`: uitgebreide Markdown-hoofdstukken.
- `snippets/`: PHP- en SQL-fragmenten per hoofdstuk.
- `assets/screenshots/`: stap-voor-stap screenshotbeelden voor boekje en Markdown.
- `opdrachten/`: losse invul- en reflectiebladen.
